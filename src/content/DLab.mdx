# DLab

### 실습실,노트북/태블릿 대여 관리 웹사이트

---

### 📝 사용 기술

- react
- flask
- huggingface
- react-router-dom
- react-fullpage
- styled-components

---

### 💡 개발 파트 / 사용 개발 툴

- FRONTEND
- figma
- [github →](https://github.com/jiyoonyee/DLab)

---

### 📖 프로젝트 개요

#### 실습실 신청과 노트북 태블릿 대여,관리를 도와주는 웹사이트 입니다.

- react를 활용해 UI를 styled-components 라이브러리를 활용해 페이지를 구현했습니다.
- flask를 이용해 각 실습실별 컴퓨터 정보와 실습실 신청정보를 axios 방식을 이용해 가져옵니다.
- huggingface에서 ai모델을 가져와 실습실 정보를 학습시켜 사용자에게 알맞은 실습실을 추천합니다.

---
